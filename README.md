## 开发
```bash
npm i
npm run dev
```

## 构建
```bash
npm run build
npm run export
```

## 环境变量

 * `STATIC_MODE`: 是否为静态构建，影响的是是否开启 Next.js 的图像优化。取值范围 `false` 或 `true`，默认为 `false`。
 * `NEXT_PUBLIC_CDN_LOCATION`: CDN 地址，默认为 `/static`。
 * `PORT`: next.js 服务端口，默认为 `3000`。
